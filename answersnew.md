## Exercise 2
### KML 
KML, or the _Keyhole Markup Language_ is an XML-based file format used for displaying geographic information. For example **Google Earth** ( for which it was originally developed in **2008**) uses KML to show features like placemarks, descriptions and paths.
### MathML
MathML, or the _Mathematical Markup Language_ is especially for the description of mathematical notations, focusing on both structure and content. It aims to integrate formulae into web pages and other online documents. It was developed in **1998** and has since become the industry standard.

public class Person{
  private string name;
  private int age;

  public Person(string initialName)
  {
    this.age = 0;
    this.name = initialName;
  }
  public void PrintPerson()
  {
    Console.WriteLine(this.name + ", age " + this.age + " years");
  }
  public void GrowOlder()
  {
    this.age = this.age + 1;
  }
}
